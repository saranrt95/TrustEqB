dataFiles folder contains the test sets used for Clopper Pearson bound calculation.
each excel file is named with the following rule: idX_N.xlsx, where X is the identifier of the model, and N is the size of the considered test set.

Specifically:

X = 1: train3000
X = 2: train2500
X = 3: train2000
X = 4: train1500
X = 5: train1000
X = 6: train9000
X = 7: train500
X = 8: train5000
